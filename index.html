Fortuna-2
A custom Z80 mini computer that fits in an Altoids tin.
  • 0.5 Mhz Z80 CPU
  • 32k RAM
  • SD Card storage
  • UART (can be connected to video and keyboard using Fortuna Terminal)

Fortuna Terminal
A VT100 terminal emulator
  • PS/2 keyboard input
  • VGA output
  • Uses two ATMEGA-328P, one to receive the keypress and the other one to generate the video
  • Once ready, it will fit into an Altoids tin
  • Based on the Minimal-Terminal project
